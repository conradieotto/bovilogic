# BoviLogic Icons

Add these icon files here:

- `icon-192.png`         – 192×192 PNG app icon (green background, white cow/B logo)
- `icon-512.png`         – 512×512 PNG app icon
- `icon-maskable-512.png`– 512×512 maskable icon (safe area: logo centered in 80% of canvas)
- `favicon.ico`          – 32×32 favicon

These are required for the PWA to work correctly.

Free icon generator: https://realfavicongenerator.net
Use color: #1B5E20 (dark green) as background.
