# Splash Screen Assets

## Photo
Drop your splash photo here as: `splash.jpg`
- Recommended size: 1080 × 1920px (portrait, mobile)
- Any JPG or PNG works — name it `splash.jpg`

## Video (optional)
Drop your splash video here as: `../videos/splash.mp4`
- Keep it short: 3–6 seconds
- Keep file size under 5MB for fast loading on mobile
- MP4 format (H.264) works on all devices
- To enable: open `index.php` and uncomment the `<video>` block, comment out the `<img>` block
